[리눅스 명령어]

1. ls : 현재 폴더내부의 파일 또는 폴더 조회
2. ls -a : 현재폴더 내부의 파일 또는 폴더 조회(숨겨진 파일 또는 폴더 포함)
3. cd : 경로 이동 (tab을 누르면 경로 자동완성 가능)
	a. 경로의 종류 : 
		- / : root 경로(최상위 경로) 
		- ./ : 현재 경로
		- ../ : 이전 경로
		- ../../ : 이전전 경로
		- ~/ : 사용자 경로
4. clear : 콘솔 내용 전체 삭제
5. pwd : 현재 폴더의 경로pubvl
6. mkdir : 폴더 생성
7. mkdir -p : 상위 폴더가 없으면 같이 생성
8. rmdir : 폴더 삭제
9. rm -r : 현재 폴더의 하위를 다 포함해서 삭제
10. cp -r : 현재 폴더를 다른 폴더 내부로 복제
11. mv : 폴더 및 파일 이동(파일 및 파일 이름 변경)새ㅕ
12. touch : 파일 생성
13. echo : 입력된 문자열들 다시 콘솔에 출력
14. echo "내용" > 파일명 : 내용을 포함한 파일 생성
15. cat : 파일 내부의 내용을 확인
